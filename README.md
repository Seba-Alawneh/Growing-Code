# Growing Code 🌿

[cite_start]A foundational Python project designed to explore core programming concepts through the lens of community garden data[cite: 5, 90]. [cite_start]This project covers variables, functions, control flow, loops, and recursion[cite: 6, 91].

[cite_start]The project is organized into exercise directories, each containing a specific Python function[cite: 22, 108]:

* [cite_start]**ex0/**: `ft_hello_garden.py` - Basic output[cite: 121].
* [cite_start]**ex1/**: `ft_plot_area.py` - Arithmetic and input[cite: 134].
* [cite_start]**ex2/**: `ft_harvest_total.py` - Summation and loops[cite: 149].
* [cite_start]**ex3/**: `ft_plant_age.py` - Conditional logic[cite: 165].
* [cite_start]**ex4/**: `ft_water_reminder.py` - Basic reminders[cite: 184].
* [cite_start]**ex5/**: `ft_count_harvest_iterative.py` & `ft_count_harvest_recursive.py` - Iteration vs Recursion[cite: 203, 205].
* [cite_start]**ex6/**: `ft_garden_summary.py` - Data formatting[cite: 240].
* [cite_start]**ex7/**: `ft_seed_inventory.py` - Type annotations and string manipulation[cite: 260, 261].
