# Growing Code 🌿

A foundational Python project designed to explore core programming concepts through the lens of community garden data. This project covers variables, functions, control flow, loops, and recursion.

## 📂 Project Structure
The project is organized into exercise directories, each containing a specific Python function:

* **ex0/**: `ft_hello_garden.py` - Basic output.
* **ex1/**: `ft_plot_area.py` - Arithmetic and input.
* **ex2/**: `ft_harvest_total.py` - Summation and loops.
* **ex3/**: `ft_plant_age.py` - Conditional logic.
* **ex4/**: `ft_water_reminder.py` - Basic reminders.
* **ex5/**: `ft_count_harvest_iterative.py` & `ft_count_harvest_recursive.py` - Iteration vs Recursion.
* **ex6/**: `ft_garden_summary.py` - Data formatting.
* **ex7/**: `ft_seed_inventory.py` - Type annotations and string manipulation.

## 🛠️ Requirements
* **Python 3.10+**.
* Code must strictly follow **flake8** linter standards.
